Módulos¶

Rotas¶

routes.main.home()[código fonte]¶

Renderiza a página inicial.

Esta função está associada à URL raiz (/) da aplicação e renderiza o template index.html.

Retorno:

Template HTML renderizado para a página inicial.

Tipo de retorno:

flask.Response

Utilitários¶

utils.helper.sample_helper_function()[código fonte]¶

Função auxiliar de exemplo.

Esta função simples retorna uma mensagem indicando que é uma função auxiliar. Pode ser usada como exemplo ou como base para outras funções auxiliares.

Retorno:

Uma string com a mensagem de exemplo.

Tipo de retorno:

str
